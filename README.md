# Netflix Data Analysis Project (EDA)

## Objective
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to understand trends in content distribution, genres, ratings, and country-wise contribution.

---

## Dataset Information
- Dataset: Netflix Movies and TV Shows  
- Source: Kaggle  
- Records: ~8800 titles  
- Features: Title, Type, Country, Rating, Duration, Date Added, Genre  

---

## Tools & Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook  

---

## Key Objectives
- Compare Movies vs TV Shows distribution  
- Analyze content addition trends over the years  
- Identify top contributing countries  
- Explore most common genres and ratings  
- Study movie duration patterns  

---

## Key Insights
- Netflix has more Movies than TV Shows  
- Content addition increased significantly after 2016  
- USA and India contribute the highest number of titles  
- Drama is the most popular genre  
- Most content is rated TV-MA (mature audience)  

---

## Conclusion
This analysis provides insights into Netflix’s content growth strategy and distribution patterns across countries, genres, and audience ratings.
